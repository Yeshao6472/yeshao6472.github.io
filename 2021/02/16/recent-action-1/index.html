最近的行为

发表于 2021-02-16 更新于 2021-02-18 本文字数： 440

最近啊，我对这个站点做了一些改动。

在腾讯云COS上部署hexo站点

https://myblog-1302602758.cos.ap-nanjing.myqcloud.com
无法用CNAME解析到这个地址，域名要先真实备案，在bucket里开启cdn加速，否则cname解析过去返回的是xml提示非法解析，或者404.

域名解析转到cloudflare

cloudflare，国外cdn提供商。

  • 免费的cdn加速与ddos防护
    因为国内腾讯云的cdn不能加速未备案的域名（不能添加），而CF可以。
    副作用：可能会导致原站访问变慢。

    作业没做完

    好家伙，还有几天开学了，作业都没做完。

    上一次的彩蛋🌏🌍🌎

    在打包的时候，发现控制台输出有一处内容。
    细心观察就会发现！
    其实就是在[python install dir]/lib/site-packages/Pyinstaller/bootloader/images下的icon-console.ico文件。
    建议先把源文件重命名为.bak后缀而不是删掉它。
